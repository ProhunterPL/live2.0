chemRxiv Submission Package - Live 2.0 Manuscript
==================================================

This package contains all files needed for chemRxiv submission.

STRUCTURE:
----------
- manuscript_arxiv.pdf          Main manuscript PDF (for direct upload)
- manuscript_arxiv.tex           LaTeX source (backup/reference)
- references.bib                 Bibliography
- figures/                       All figure files (PNG, 300 DPI)
- tables/                        All table files (LaTeX format)

SUBMISSION INSTRUCTIONS:
------------------------
1. Go to: https://chemrxiv.org/engage/chemrxiv/login
2. Click "Submit a Preprint"
3. Upload: manuscript_arxiv.pdf (main file)
4. Fill in metadata (title, abstract, keywords, authors)
5. Submit

NOTE: chemRxiv primarily accepts PDF files. The LaTeX source, figures, 
and tables are included here as backup/reference only.

For detailed instructions, see: CHEMRXIV_SUBMISSION_GUIDE.md
